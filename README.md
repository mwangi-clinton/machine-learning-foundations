# machine-learning-foundations

Welcome to my machine learning foundations repository! This space is dedicated to experimenting, learning, and getting my hands dirty with classical machine learning algorithms.

Rather than just calling libraries as black boxes, the goal of this project is to build a strong intuition for how these models actually work under the hood—bridging the gap between the underlying mathematics and practical, working code.

### Goals of this Repository

- **Experimentation:** Testing how algorithms respond to different hyperparameters, noise levels, and data distributions.
- **Hands-on Learning:** Implementing algorithms from scratch and visualizing their mechanics (like gradient descent and loss landscapes) using interactive notebooks.
- **Building Intuition:** Developing a deep, foundational understanding of the math and logic that powers classical ML.
